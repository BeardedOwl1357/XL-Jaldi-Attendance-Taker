Attendance without calling 400 names

Load the roster, listen row by row, and let the app turn announced roll numbers into a live present list with absentees generated automatically.

Recommended flow

  1. Start with the full roster.
  2. Students announce only their own roll numbers.
  3. Operator enters each roll quickly.
  4. Everyone not checked in becomes absent.
